# UnityLuaFramework
一个用LUA写的框架游戏框架

我把基本上所有的业务多删除了。 就保留了一个最基本的背包业务，可以用来研究逻辑层的写法。

这套框架运行是非常良好的， 拥有多层级的PROXY， 开发效率，运行效率都比较的高。
